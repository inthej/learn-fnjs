# 제너레이터/이터레이터 : 제너레이터: 이터레이터이자 이터러블을 생성하는 함수이다 # 제너레이터 특징 - 제너레이터는 Symbol.iterator 를 가지고 있다. - 제너레이터는 well-formed 이터레이터를 리턴하는 함수이다 - 제너레이터는 리턴값을 가질 수 있지만.. 리턴값은 순회대상 값은 아니고.. 이터레이터 값이 done: true 가 될적에 발생된느 값이다 - 제너레이터는 순회할 값을 문장으로 표현하는 것이다 라고 말 할수 있다.. - 제너레이터를 통해서 자바스크립트에서는 사실상.. 어떠한 상태나 어떠한 값이든 사실상 순회할 수 있도록 만들 수 있다..
